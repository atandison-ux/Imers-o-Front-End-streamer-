🎬 Netflix Clone - Imersão Front-End

Este projeto foi desenvolvido durante a Imersão Front-End e tem como objetivo recriar a interface da Netflix utilizando HTML, CSS e JavaScript.  
O foco é praticar conceitos de responsividade, estilização moderna e manipulação de elementos visuais.

---

🚀 Tecnologias utilizadas
- HTML5 → Estrutura da aplicação  
- CSS3 → Estilização e responsividade  
- JavaScript (ES6+) → Interatividade e lógica  
- Git/GitHub → Controle de versão e hospedagem  

---

📂 Estrutura do projeto
`
Netflix/
│── index.html        # Página principal
│── style.css         # Estilos globais
│── script.js         # Lógica e interatividade
│── assets/           # Imagens e ícones
`

---

🎯 Funcionalidades
- Layout inspirado na Netflix  
- Seções de filmes e séries organizadas por categorias  
- Design responsivo para diferentes dispositivos  
- Interatividade com JavaScript  

---

🖥️ Como executar
1. Clone este repositório:
   `bash
   git clone https://github.com/atandison-ux/Imers-o-Front-End-streamer-.git
   `
2. Acesse a pasta do projeto:
   `bash
   cd Imers-o-Front-End-streamer-/Netflix
   `
3. Abra o arquivo index.html em seu navegador.
